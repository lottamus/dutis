# Dutis

A command-line tool to select default applications. It is a wrapper around [duti](https://github.com/moretension/duti).

## Installation

```shell
$ go install github.com/tsonglew/dutis@latest

## OR

brew tap tsonglew/dutis
brew install dutis
```

## Usage

```shell
$ dutis
```
